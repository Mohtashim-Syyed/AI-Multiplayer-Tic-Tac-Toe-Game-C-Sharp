# AI-Multiplayer-Tic-Tac-Toe Game - C# using MinMax Algorithm
Tic-Tac-Toe game written in C#, built as a simple console-based strategy board game that’s easy to understand and play. It follows the traditional rules of tic-tac-toe and includes both multiplayer functionality and an AI opponent.

After starting the game, a console screen will appear with a tic-tac-toe board, other rules are the same. First, the player has to select whether he want to take first turn or he want to give it to AI. It contains the Minimax algorithm, which is a decision rule used for a two-player game. The gameplay design is so simple that user won’t find it difficult to use and understand.
